It seems like power shell is corrupted somehow,, to restore and scan windows system files, type in start menu "CMD" and run it as an administrator
then run this command:
sfc /scannow

After it finishes, run this command in the same window:

dism /online /cleanup-image /restorehealth

restart your device afterwards and see if it helps
